I have created the standalone Python script `e2b_security_task.py` which performs the requested sequential operations using the E2B Python SDK.

The script:
1.  Creates an E2B Sandbox with a timeout of 600 seconds.
2.  Creates `/home/user/scan_network.sh` inside the sandbox with the exact bash script content provided.
3.  Makes the bash script executable using `chmod +x`.
4.  Executes the script and captures its `stdout` and `stderr`.
5.  Writes the captured outputs to `/home/user/scan_results.txt` inside the sandbox in the specified format.
6.  Saves the sandbox ID to `/home/user/e2b_task_info.json` locally.

The script content is as follows:

```python
import json
from e2b import Sandbox

def main():
    # 1. Create an E2B Sandbox with a timeout of 600 seconds
    with Sandbox(timeout=600) as sb:
        # 2. Inside the sandbox, create a bash script at /home/user/scan_network.sh
        bash_script_content = """#!/bin/bash
echo 'Scanning open ports...'
echo 'Found vulnerable port 8080' >&2
echo 'Scan complete. 1 vulnerabilities found.'
"""
        sb.files.write("/home/user/scan_network.sh", bash_script_content)

        # 3. Change permissions of /home/user/scan_network.sh to make it executable
        sb.commands.run("chmod +x /home/user/scan_network.sh")

        # 4. Execute the script /home/user/scan_network.sh inside the sandbox
        # 5. Capture the stdout and stderr from the command execution
        execution = sb.commands.run("/home/user/scan_network.sh")
        stdout = execution.stdout
        stderr = execution.stderr

        # 6. Write a file inside the sandbox at /home/user/scan_results.txt
        # The format requires STDOUT: followed by captured stdout, then STDERR: followed by captured stderr.
        results_content = f"STDOUT:\n{stdout}STDERR:\n{stderr}"
        sb.files.write("/home/user/scan_results.txt", results_content)

        # 7. Save the created sandbox ID to a local file at /home/user/e2b_task_info.json
        task_info = {"sandbox_id": sb.id}
        with open("/home/user/e2b_task_info.json", "w") as f:
            json.dump(task_info, f)

if __name__ == "__main__":
    main()
```
