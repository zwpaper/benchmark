I have created the Python script `solve.py` which uses the E2B SDK to spin up a web server inside a sandbox and expose it via a public URL.

The script performs the following actions:
1.  **Creates a new E2B sandbox** using `Sandbox.create()`.
2.  **Writes an `index.html` file** to `/home/user/index.html` inside the sandbox with the content "Hello from E2B Sandbox!".
3.  **Starts a Python HTTP server** on port 8000 in the background inside the sandbox, serving the `/home/user/` directory.
4.  **Retrieves the public URL** for port 8000 using `sandbox.get_host(8000)`.
5.  **Saves the sandbox ID and URL** to a local JSON file at `/home/user/e2b_task_info.json`.
6.  **Keeps the sandbox alive** for 5 minutes (300 seconds) to allow for verification of the URL.

The script requires the `E2B_API_KEY` environment variable to be set to authenticate with the E2B service.

```python
import json
import time
import os
from e2b import Sandbox

def solve():
    # 1. Creates a new E2B sandbox using the `e2b` SDK.
    print("Creating E2B sandbox...")
    # Note: Ensure E2B_API_KEY environment variable is set.
    sandbox = Sandbox.create()

    try:
        sandbox_id = sandbox.sandbox_id
        print(f"Sandbox created with ID: {sandbox_id}")

        # 2. Writes an `index.html` file in `/home/user/` containing the exact text `Hello from E2B Sandbox!`.
        content = "Hello from E2B Sandbox!"
        print("Writing index.html...")
        sandbox.files.write("/home/user/index.html", content)

        # 3. Starts a Python HTTP server in the background inside the sandbox on port 8000, serving the `/home/user/` directory.
        print("Starting Python HTTP server on port 8000...")
        # We start the server in the background and specify /home/user as the working directory.
        sandbox.commands.run("python3 -m http.server 8000", background=True, cwd="/home/user")

        # 4. Retrieves the public URL for port 8000 using the SDK.
        # get_host(port) returns the hostname for the specified port.
        hostname = sandbox.get_host(8000)
        url = f"https://{hostname}"
        print(f"Public URL: {url}")

        # 5. Saves the sandbox ID and the full public URL (e.g. `https://<host>`) as a JSON object in `/home/user/e2b_task_info.json` on the local machine.
        task_info = {
            "sandbox_id": sandbox_id,
            "url": url
        }
        
        # The local path where the task info will be saved.
        output_file = "/home/user/e2b_task_info.json"
        with open(output_file, "w") as f:
            json.dump(task_info, f, indent=4)
        print(f"Task info saved to {output_file}")

        # 6. Ensure the sandbox is kept alive for at least 5 minutes so we can verify the URL.
        print("Keeping sandbox alive for 5 minutes (300 seconds) to allow verification...")
        time.sleep(300)

    finally:
        # After 5 minutes, we clean up the sandbox.
        print("Closing sandbox...")
        sandbox.kill()

if __name__ == "__main__":
    solve()
```
