I have created the Python script `/home/user/provision.py` that automates the provisioning of an E2B sandbox.

The script performs the following actions:
1.  **Creates a new E2B sandbox** using `Sandbox.create(timeout=3600)`, ensuring it stays alive for an hour as requested.
2.  **Creates the directory** `/etc/custom_nginx/` within the sandbox.
3.  **Writes the specified Nginx configuration** to `/etc/custom_nginx/nginx.conf`.
4.  **Reads back the file** to verify that the content was correctly written.
5.  **Writes the `sandbox_id`** to a JSON file at `/home/user/e2b_task_info.json` on the host machine.

The script assumes the `E2B_API_KEY` is already set in the environment.

### File: `/home/user/provision.py`
```python
import json
import os
from e2b_code_interpreter import Sandbox

def provision_sandbox():
    # 1. Create a new sandbox environment
    # Setting a long timeout (e.g., 1 hour = 3600 seconds) to keep it alive as requested
    print("Creating sandbox...")
    sandbox = Sandbox.create(timeout=3600)
    sandbox_id = sandbox.sandbox_id
    print(f"Sandbox created with ID: {sandbox_id}")

    try:
        # 2. Create directory /etc/custom_nginx/
        print("Creating directory /etc/custom_nginx/...")
        # Note: In E2B, sandbox.files.make_dir creates directories recursively if needed
        sandbox.files.make_dir("/etc/custom_nginx")

        # 3. Write configuration file /etc/custom_nginx/nginx.conf
        nginx_conf_content = """server {
    listen 8080;
    server_name localhost;
    location / {
        root /var/www/html;
        index index.html;
    }
}"""
        print("Writing nginx.conf...")
        sandbox.files.write("/etc/custom_nginx/nginx.conf", nginx_conf_content)

        # 4. Read the file back to verify content
        print("Verifying file content...")
        read_content = sandbox.files.read("/etc/custom_nginx/nginx.conf")
        
        if read_content == nginx_conf_content:
            print("Verification successful: Content matches.")
        else:
            print("Verification failed: Content does not match!")

        # 5. Write sandbox_id to /home/user/e2b_task_info.json on the host
        host_info_path = "/home/user/e2b_task_info.json"
        info_data = {
            "sandbox_id": sandbox_id
        }
        with open(host_info_path, "w") as f:
            json.dump(info_data, f, indent=4)
        print(f"Sandbox info saved to {host_info_path}")

    except Exception as e:
        print(f"An error occurred: {e}")
        # Not closing the sandbox as per requirement "keep the sandbox alive"
        raise

if __name__ == "__main__":
    provision_sandbox()
```   
