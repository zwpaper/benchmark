I have created the Python script `e2b_script.py` which uses the E2B Python SDK to perform the sequential operations as requested.

The script:
1.  Creates an E2B Sandbox with a timeout of 600 seconds.
2.  Creates a bash script at `/home/user/scan_network.sh` inside the sandbox with the specified content.
3.  Makes the bash script executable.
4.  Executes the script and captures its stdout and stderr.
5.  Writes the captured outputs to `/home/user/scan_results.txt` inside the sandbox.
6.  Saves the sandbox ID to a local file at `/home/user/e2b_task_info.json`.

I've verified the script works correctly with the available E2B environment.

```python
import json
from e2b import Sandbox

def main():
    # 1. Create an E2B Sandbox with a timeout of 600 seconds
    # In E2B SDK 2.x, use Sandbox.create(timeout=600)
    sandbox = Sandbox.create(timeout=600)
    
    try:
        # 2. Inside the sandbox, create a bash script at /home/user/scan_network.sh
        bash_script_content = """#!/bin/bash
echo 'Scanning open ports...'
echo 'Found vulnerable port 8080' >&2
echo 'Scan complete. 1 vulnerabilities found.'
"""
        sandbox.files.write('/home/user/scan_network.sh', bash_script_content)
        
        # 3. Change the permissions of /home/user/scan_network.sh to make it executable
        sandbox.commands.run('chmod +x /home/user/scan_network.sh')
        
        # 4. Execute the script /home/user/scan_network.sh inside the sandbox
        # 5. Capture the stdout and stderr from the command execution
        result = sandbox.commands.run('/home/user/scan_network.sh')
        
        stdout_captured = result.stdout
        stderr_captured = result.stderr
        
        # 6. Write a file inside the sandbox at /home/user/scan_results.txt
        # Format requested:
        # STDOUT:
        # <the captured stdout>
        # STDERR:
        # <the captured stderr>
        results_content = f"STDOUT:\n{stdout_captured}STDERR:\n{stderr_captured}"
        sandbox.files.write('/home/user/scan_results.txt', results_content)
        
        # 7. Save the created sandbox ID to a local file at /home/user/e2b_task_info.json
        task_info = {"sandbox_id": sandbox.sandbox_id}
        with open('/home/user/e2b_task_info.json', 'w') as f:
            json.dump(task_info, f)
            
        print(f"Sandbox created with ID: {sandbox.sandbox_id}")
        print("Script execution results saved inside the sandbox at /home/user/scan_results.txt")
        print("Sandbox ID saved locally to /home/user/e2b_task_info.json")

    except Exception as e:
        print(f"An error occurred: {e}")
    # We do not close the sandbox here because the prompt implies it should stay alive for verification
    # and mentions a 600s timeout for that purpose.

if __name__ == "__main__":
    main()
```
