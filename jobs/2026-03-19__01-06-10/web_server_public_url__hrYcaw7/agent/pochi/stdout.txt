- **File Writing:** Writes `index.html` with the text `Hello from E2B Sandbox!` to `/home/user/` inside the sandbox.
- **Server Execution:** Starts a Python HTTP server on port 8000 in the background, serving the `/home/user/` directory.
- **URL Retrieval:** Uses `sandbox.get_host(8000)` to get the host and prefixes it with `https://` to form the full public URL.
- **Task Info Storage:** Saves the `sandbox_id` and `url` to a local file `/home/user/e2b_task_info.json`.
- **Sandbox Persistence:** Keeps the sandbox alive for 5 minutes using `time.sleep(300)` inside the `with` block.

### `solve.py` content:
```python
import json
import time
import os
from e2b import Sandbox

def main():
    # Set the sandbox timeout to 300 seconds (5 minutes)
    print("Creating E2B sandbox...")
    # Use the 'with' statement to manage the sandbox lifecycle
    with Sandbox.create(timeout=300) as sandbox:
        sandbox_id = sandbox.sandbox_id
        print(f"Sandbox created with ID: {sandbox_id}")

        # 1. Write index.html file in /home/user/ inside the sandbox
        index_content = "Hello from E2B Sandbox!"
        print(f"Writing index.html to sandbox at /home/user/index.html...")
        sandbox.files.write("/home/user/index.html", index_content)

        # 2. Start Python HTTP server in the background inside the sandbox on port 8000
        # Serving the /home/user/ directory
        print("Starting Python HTTP server on port 8000...")
        # Use background=True to run the server in the background
        sandbox.commands.run("python3 -m http.server 8000 --directory /home/user/", background=True)

        # 3. Retrieve the public URL for port 8000 using the SDK
        # Wait a moment to ensure the server is up and the port is detected
        time.sleep(2)
        host = sandbox.get_host(8000)
        # The host returned by get_host is the domain name, e.g. 8000-xxx.e2b.app
        public_url = f"https://{host}"
        print(f"Public URL: {public_url}")

        # 4. Save the sandbox ID and the full public URL to e2b_task_info.json on the local machine
        task_info = {
            "sandbox_id": sandbox_id,
            "url": public_url
        }
        local_json_path = "/home/user/e2b_task_info.json"
        print(f"Saving task info to {local_json_path} on local machine...")
        with open(local_json_path, "w") as f:
            json.dump(task_info, f, indent=2)

        # 5. Ensure the sandbox is kept alive for at least 5 minutes
        print("Keeping sandbox alive for 5 minutes (300 seconds)...")
        time.sleep(300)

if __name__ == "__main__":
    main()
```
