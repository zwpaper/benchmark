I have created the `solve.py` script as requested. The script uses the E2B SDK to create a sandbox, write an `index.html` file, start a Python HTTP server on port 8000, and save the sandbox ID and public URL to a local JSON file. It also ensures the sandbox is kept alive for at least 5 minutes.

### `solve.py`
```python
import json
import time
from e2b import Sandbox

def main():
    # 1. Create a new E2B sandbox
    print("Creating E2B sandbox...")
    sandbox = Sandbox.create()
    
    try:
        # 2. Write index.html in /home/user/
        print("Writing index.html...")
        sandbox.files.write("/home/user/index.html", "Hello from E2B Sandbox!")
        
        # 3. Start Python HTTP server on port 8000 in background
        print("Starting HTTP server...")
        # We use background=True to run it in the background
        # We specify cwd="/home/user" to serve that directory
        sandbox.commands.run("python3 -m http.server 8000", background=True, cwd="/home/user")
        
        # Ensure the sandbox doesn't timeout for at least 5 minutes
        sandbox.set_timeout(300)
        
        # 4. Retrieve the public URL for port 8000
        # get_host(8000) returns the hostname for the given port
        hostname = sandbox.get_host(8000)
        url = f"https://{hostname}"
        print(f"Sandbox ID: {sandbox.sandbox_id}")
        print(f"Public URL: {url}")
        
        # 5. Save sandbox ID and public URL as JSON locally
        info = {
            "sandbox_id": sandbox.sandbox_id,
            "url": url
        }
        local_path = "/home/user/e2b_task_info.json"
        with open(local_path, "w") as f:
            json.dump(info, f, indent=2)
        print(f"Saved info to {local_path}")
        
        # 6. Keep the sandbox alive for at least 5 minutes
        print("Keeping sandbox alive for 5 minutes...")
        # We use time.sleep(300) to keep the script running, 
        # which keeps the sandbox object and connection active.
        time.sleep(300)
        
    finally:
        # The sandbox will be killed when the script ends or when killed explicitly.
        # Since we want it to stay alive for 5 minutes, the sleep above handles it.
        print("Done.")

if __name__ == "__main__":
    main()
```

The script expects the `E2B_API_KEY` environment variable to be set, which it is in the current environment. You can run it with:
```bash
python3 solve.py
```
